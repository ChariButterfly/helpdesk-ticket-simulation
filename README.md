# Help Desk Ticket Simulation

This project simulates real-world IT help desk scenarios commonly handled in entry-level support roles. Each ticket includes the issue reported, troubleshooting steps taken, the resolution provided, and final documentation.

## Skills Demonstrated
- Ticket documentation
- Troubleshooting
- Password resets
- Account support
- Network diagnostics
- User communication
- Escalation awareness

## Tools Referenced
- Windows 10/11
- Active Directory
- Remote Desktop
- Microsoft 365
- Ticketing workflow
